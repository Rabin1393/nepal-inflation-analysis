# Nepal Inflation & Price Trend Analyzer 🇳🇵

## 📌 Overview
This project analyzes price trends of essential goods in Nepal to study inflation patterns across cities using Python and data analysis tools.

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📊 Features
- City-wise price comparison
- Item-wise inflation trends
- Time-series analysis
- Heatmap visualization

## 📈 Key Insights
- Price variation across Kathmandu, Lalitpur, and Bhaktapur
- Rising inflation trend in essential goods
- Oil prices increase faster than rice

pip install pandas seaborn matplotlib
